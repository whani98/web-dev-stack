1. 전체 선택자 : *

2. 태그 선택자 : 태그

해당 태그에 스타일을 적용

3. 클래스 선택자 : .클래스명

class 1 스타일 적용

class 2 스타일 적용

4. 아이디 선택자 : #아이디명

id1 스타일 적용

5. 그룹 선택자

콤마(,)를 사용하여 여러 요소 선택

6. 자식/자손 선택자
